Manual Técnico: Academia Musical JACQUIN 🎶

Todo lo que necesitas saber para gestionar y entender el sistema.

1. ¿Qué es este proyecto?

Es la plataforma digital de la Academia Musical JACQUIN. No es solo una web; es un sistema que conecta a estudiantes, profesores y administradores para que la música no pare.

2. Las piezas del motor (Arquitectura)

  • La Web (Frontend): HTML, CSS y JS puro. Rápida y adaptable.
  • La API (Backend): PHP. Gestiona datos, correos y seguridad.
  • Base de Datos: MySQL para almacenamiento seguro.

3. Instalación

  1. Descarga XAMPP con PHP 8+.
  2. Copia el proyecto en htdocs/.
  3. Importa el SQL de jacquin_api/database/.
  4. Configura jacquin_api/config/env.php.
  5. Abre en tu navegador: localhost/project_jacquin/web_page.

4. Estructura

📂 Web Page

  • /pages: Vistas HTML.
  • /js: Lógica dinámica.
  • /css: Diseño premium.

📂 Jacquin API

  • /config: Llaves del sistema.
  • admin_*.php: Gestión administrativa.

5. Flujos Clave

Gestión de eventos, inscripciones seguras y mantenimiento con túneles Zrok para acceso remoto.
