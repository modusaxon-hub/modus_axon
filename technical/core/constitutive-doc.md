# MODUS AXON: Acta de Constitución Técnica (Core Software)
ID: MA-CORE-001

## 1. Naturaleza del Sistema
MODUS AXON no es solo una empresa matriz, es el **Núcleo Orquestador (Core)** que gobierna la lógica, seguridad y documentación de todo el ecosistema de proyectos ADSO.

## 2. Arquitectura de Gobernanza
El sistema opera mediante un modelo de **Matriz y Satélites**:
- **Matriz (MODUS AXON)**: Repositorio central de verdad técnica, bitácoras cronológicas e inteligencia de negocio.
- **Satélites (Proyectos)**: Instancias operativas ejecutables que se sincronizan con la matriz mediante el `modus-connector.md`.

## 3. Pilares del Software Matriz
- **Trazabilidad Universal**: Cada acción técnica en cualquier satélite genera un ID único (`MA-PROJ-ID-XXX`) en la bitácora central.
- **Seguridad Distribuida**: Las políticas de seguridad se definen en el núcleo y se heredan a los proyectos.
- **IA Concatenada**: Los agentes (Antigravity) operan bajo el `work-master-flow.md` alojado en el core.

## 4. Stack de Gobernanza
- **Orquestación**: Agentes Antigravity + Skills.
- **Conocimiento**: NotebookLM Map Integration.
- **Infraestructura**: Supabase Matrix Cloud.
